### 📊 NABFID HR Dashboard (Infreyx Platform)

This project showcases an end-to-end **HR Analytics Dashboard** built for the NABFID use case using the **Infreyx platform**. The goal of this project is to convert raw HR data into meaningful insights while ensuring strong data quality and governance.

The project covers the complete data lifecycle — from data ingestion to visualization — enabling efficient workforce analysis and decision-making.

---

### 🔄 Project Workflow

1. **Data Ingestion**

   * Imported raw HR data using **Infreyx Data Workbench**
   * Managed and configured data sources within the platform

2. **Data Engineering**

   * Performed data transformation and processing using Infreyx Data Engineering
   * Cleaned and structured the data for analysis

3. **Dataset Creation**

   * Created curated datasets from processed data
   * Ensured data consistency and usability for analytics

4. **Data Quality Implementation**

   * Applied Data Quality (DQ) rules
   * Used severity levels (INFO, HIGH) and threshold checks
   * Identified anomalies and ensured reliable data

5. **Visualization (Vizpod)**

   * Built visualizations using Infreyx Data Analytics (Vizpod)
   * Created charts for HR metrics like hiring trends, department analysis, etc.

6. **Dashboard Creation**

   * Combined multiple vizpods into an interactive HR dashboard
   * Enabled easy monitoring of key HR insights

### 🔧 Key Features

* 📌 Employee joining trends analysis
* 📌 Department-wise employee distribution
* 📌 Salary and compensation insights
* 📌 Data Quality monitoring (rules, thresholds, severity)
